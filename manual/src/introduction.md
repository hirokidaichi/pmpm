# はじめに

pmpm は CLI ファーストのプロジェクトマネジメント基盤です。人と AI エージェントのどちらでも使いやすいように、`pmpm <resource> <verb>` の一貫した構文と機械可読な出力を提供します。

このマニュアルでは、プロジェクトマネジメントの実務で pmpm をどう使うかに焦点を当てます。特に、以下を重視します。

- 実際の運用シナリオを通じた手順
- タスク、依存関係、マイルストーン、リスク、時間管理の使い分け
- チーム運用での最小限のルールとコツ

注意: pmpm は開発初期段階のため、コマンドや挙動は変わる可能性があります。運用に組み込む際は、まず小さなプロジェクトで試すことを推奨します。
